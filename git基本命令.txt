
##更新代码
git fetch origin master  从远程的origin创库的master分支下载代码到本地的origin master
git merge origin/master 将远程master分支下载下来的代码合并到本地master

##提交代码
git add . 递归的添加当前目录中的所有改动或新增的文件
git commit -m 'commit说明' 提交add操作的文件
git push origin 1.3.0-feature 将commit的文件推送到远程git服务器

##创建分支
git checkout -b 1.0.0-feature 创建1.0.0-feature分支
git push 1.0.0-feature   将刚刚创建的分支发布到远程
git pull之后可以看到别人创建的新分支

##删除分支
git branch -d 1.3.0-feature 删除本地1.3.0-feature分支
git push origin --delete 1.3.0-feature 删除远程1.3.0-feature分支
git remote show origin 来查看分支的状态
git remote prune origin 将不存在的分支删除掉

##切换分支
git checkout master

##重命名本地分支
git branch -m 1.4.0-feature 1.4.0-renamed-feature
